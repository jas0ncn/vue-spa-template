# Vue SPA Template

> A single page application template for vuejs

## Build Setup

``` bash
# init 
git clone git@github.com:jas0ncn/vue-spa-template.git

# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm test
```
